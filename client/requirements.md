## Packages
framer-motion | Window dragging, animations, and transitions
recharts | System monitor graphs (CPU/Memory usage)
date-fns | Date formatting for chat messages and system logs
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
The application simulates a desktop OS.
Chat uses SSE (Server-Sent Events) for streaming responses.
Background needs to be a dark, animated sci-fi grid or nebula.
